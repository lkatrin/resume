Приложение для ведения расписания электричек.
Реализовано с помощью PyQT6 и sqlite3.

Функционал:

- авторизация и регистрация пользователей
- удаление пользователей

- добавление/редактирование/удаление поездов из списка, для поездов вводятся такие
данные как название, время отправления и прибытия, маршрут, статус (через выпадающий список),
расписание по дням (через выпадающий список), направление (через выпадающий список),
номер платформы и пути
- диаграмма статусов для визуального отображения соотвествия электричек расписанию
- поиск по направлению (выбор через выпадающий список), по дате (выбор через визуальный календарь),
также реализована поисковая строка
- экспорт таблицы расписания в Excel
