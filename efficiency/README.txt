Приложение для оценки эффективности сотрудников. Сделано под формат мобильного.
Реализовано с помощью kivy и sqlite3.

Приложение обладает следующими функциональными возможностями:
• Регистрация с разграничением прав пользователей (администратор/пользователь), авторизация;
• Восстановление пароля по секретному вопросу;

Для админа:
• Добавление в список информации о сотруднике с его должностью:
• Редактирование/удаление информации о сотруднике;
• Оценивание работы сотрудника по нескольким критериям;
• Анализ рассчитанной эффективности для всех сотрудников;
Для пользователя:
• Вывод истории всех оценок для выбранного сотрудника;
• Вывод рассчитанной текущей и прошлой эффективности.
